# 📱 Smart Gate Pass System

## 📌 Overview

The Smart Gate Pass System is an Android-based application designed to digitize the traditional gate pass process in colleges. It reduces manual paperwork and speeds up approval using a digital workflow and QR-based verification.

## 🎯 Objective

To automate the gate pass system and improve security by replacing manual approval with a real-time digital system.

## 🚀 Features

* Students can request gate pass with reason and phone number
* Faculty can approve or reject requests
* Approved requests are forwarded to HOD
* HOD gives final approval
* QR code generated with 10-minute validity
* Security scans QR code at the gate

## 🛠️ Technologies Used

* Android (Java/Kotlin)
* Firebase / Backend API
* QR Code Generator & Scanner

## ⚙️ How It Works

1. Student submits gate pass request
2. Faculty reviews and approves/rejects
3. If approved, request goes to HOD
4. HOD gives final approval
5. QR code is generated for the student
6. Security scans QR code before exit
